参赛对象

佛山市范围内的中等职业院校、高等职业院校。

参赛方式

登录

登录大赛官方指定网站“微课慕课网-佛山市职业院校微课教学大赛（第一届）”（http://fsds.wkmk.com）首页，在首页右上角点击“注册”（如已在“微课慕课网”注册过的教师用户， 无须重新注册），输入您的个人注册信息，并选择您所在“学校”，如实填写您的手机和邮箱地址，以便获奖后能联系上您，提交信息并完成注册。

上传作品

参赛教师在大赛官方平台登录后即可上传大赛微课作品，须在大赛作品上传开放时间内上传作品，上传参赛作品详细操作步骤和注意事项请看大赛平 台底部“上传参赛作品须知”。也可以加入“微课慕课网”官方交流QQ群：282673318。

大赛要求

教学视频要求

图像清晰稳定、构图合理、声音清楚，能较全面真实反映教学情境，能充分展示教师良好教学风貌。视频片头应显示“课程名称—微课（知识点或技能点等）标题、作者和单位”，主要教学环节有字幕 提示。视频格式及上传要求详见官方网站平台中作品提交要求。

演示文稿要求

演示文稿限定为PPT格式。要求围绕教学目标，反映主要教学内容，与教学视频合理搭配；其他与教学内容相关辅助材料，如练习测试、教学评价、多媒体素材（视频、动画、图片等材料）也在大赛平 台中以微课配套的微资源的方式上传，格式符合大赛平台上传要求。

教学设计要求

教学设计应反映教师教学思想、课程设计思路和教学特色，包括教学背景、教学目标、教学方法和教学总结等方面内容，并在开头注明讲课内容所属大类专业（2位代码）、专业（4位代码）、课程名称、 知识点（技能点）名称及适用对象等信息。文件格式：doc或者docx，在大赛平台中以微课配套的微资源的方式上传。

个人奖

大赛设一、二、三等奖及优秀奖，分别予以奖励。

优秀组织奖

面向各参赛单位（职业院校）和各区赛事组织单位，根据赛事组织情况，评选优秀组织奖，分别予以奖励。

事项一

参赛者享有作品的著作权，参赛者须同意授权赛事主办方与微课慕课网享有网络传播权。所有参赛作品向社会免费开放，主办方授权相关单位享有专属出版权，出版后，原创者有署名权及获得报酬权。

事项二

参赛作品及材料需为本人原创，不得抄袭他人作品，侵害他人版权，若发现参赛作品侵犯他人著作权，或有任何不良信息内容，则一律取消参赛资格。

事项三

赛事主办单位不向参赛教师或学校收取决赛评审费用。
